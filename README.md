# Conversor de Monedas 

Aplicación de consola en Java que convierte monedas en tiempo real
usando la API de ExchangeRate-API.

## 🛠 Tecnologías utilizadas
- Java 17
- Biblioteca Gson 2.13.2
- ExchangeRate-API
- IntelliJ IDEA

##  Monedas disponibles
| Código | Moneda |
|--------|--------|
| USD | Dólar estadounidense |
| ARS | Peso argentino |
| BRL | Real brasileño |
| COP | Peso colombiano |

## ⚙️ Funcionalidades
- Conversión en tiempo real entre 6 pares de monedas
- Menú interactivo con opciones numéricas
- Historial de conversiones con marca de tiempo
- Manejo de errores de entrada y conexión

## 🚀 Cómo ejecutar
1. Clona el repositorio
2. Abre el proyecto en IntelliJ IDEA
3. Añade tu API Key en `ConversorMonedas.java`
4. Ejecuta `Main.java`

## 📌 Ejemplo de uso
```
******************************************
  Bienvenido/a al Conversor de Moneda
******************************************

1) Dólar           =>  Peso argentino
2) Peso argentino  =>  Dólar
3) Dólar           =>  Real brasileño
4) Real brasileño  =>  Dólar
5) Dólar           =>  Peso colombiano
6) Peso colombiano =>  Dólar
7) Ver historial de conversiones
8) Salir
```
